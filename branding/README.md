# FarmHand PRO Branding Library

This folder contains every official branding asset used throughout FarmHand PRO.

## Folder Structure

### 01-Logo
Official logos in every format.

### 02-Hero
Dashboard, Login and Splash backgrounds.

### 03-Colors
Official colour palette and theme files.

### 04-Typography
Fonts and typography guidelines.

### 05-Icons
Application icon library.

### 06-Mockups
Website, dashboard, clothing, vehicle and mobile mockups.

### 07-Marketing
Marketing graphics, screenshots and launch material.

### 08-BrandGuide
Official FarmHand PRO Brand Guidelines.

Only approved branding assets should be stored here.
